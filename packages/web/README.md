# @opencvjs/web

[OpenCV.js](https://docs.opencv.org/4.x/d0/d84/tutorial_js_usage.html) for browser.

## Installation

```bash
npm install @opencvjs/web
```

## Usage

```ts
import { load } from "@opencvjs/web";
```

## TypeScript Definitions

The definitions are based on the [@techstark/opencv-js](https://github.com/TechStark/opencv-js) and [mirada](https://github.com/cancerberoSgx/mirada) projects.
